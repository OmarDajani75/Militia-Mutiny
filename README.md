# Militia-Mutiny
Java tank game project done for CSC413

CONTROLS:
| Command       | Player 1    | Player 2 |
|---------------|-------------|----------|
|  Forward      | Up arrow    | W        |
|  Backward     | Down arrow  | S        |
|  Rotate left  | Left arrow  | A        |
|  Rotate Right | Right arrow | D        |
|  Shoot        | Spacebar    | F        |
